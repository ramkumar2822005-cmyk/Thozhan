# **Project Thozhan**
# 🌾 AI-Based Crop Intelligence System

An end-to-end **Data Science & Machine Learning project** that predicts crop yield, production, demand, and pricing, along with an interactive analytics dashboard for agricultural insights.

---

## 🚀 Features

- 🔐 User Authentication (Login/Register with bcrypt)
- 🌱 Crop Yield Prediction using ML models
- 🌧 Rainfall Prediction based on sowing & harvest dates
- 📊 Demand & Price Prediction
- 🧮 Production Estimation using area & yield
- 📈 Interactive Analytics Dashboard (Plotly)
- 🗄 Cloud Database Integration (MySQL - Railway)
- 👨‍🌾 Farmer Registration & Tracking
- 🧑‍💼 Admin Panel for monitoring system data

---

## 🧠 Machine Learning Models

- Random Forest Regressor  
- XGBoost Regressor  
- Regression-based prediction models  
- Custom demand estimation logic  

---

## 📊 Data Science Workflow

1. Data Collection from multiple sources  
2. Data Cleaning & Preprocessing  
3. Feature Engineering (crop, rainfall, population)  
4. Model Training & Evaluation (R², MAE, MSE)  
5. Prediction & Visualization  

---

## 🛠 Tech Stack

- **Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn, XGBoost, Plotly  
- **Framework:** Streamlit  
- **Database:** MySQL (Railway Cloud)  
- **Authentication:** bcrypt  

---

## 🖥️ Application Modules

### 🔹 Prediction & Registration
- Input crop, district, area, and sowing date  
- Get predictions: yield, production, demand, price  

### 🔹 Dashboard Analytics
- Crop-wise & district-wise production  
- Farmer activity tracking  
- Interactive visualizations  

### 🔹 Admin Panel
- View all registered data  
- Monitor system usage
